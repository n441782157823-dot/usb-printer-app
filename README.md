# اپ پرینتر (Printer App)

اپلیکیشنی برای پرینت کردن فایل‌ها از طریق اتصال USB OTG

## ویژگی‌ها

✅ انتخاب فایل از گالری و فایل‌منیجر  
✅ پشتیبانی از فرمت‌های مختلف (PDF، عکس، Word، TXT)  
✅ پرینت مستقیم از طریق USB OTG  
✅ رابط کاربری سادگی و فارسی‌پذیر  
✅ ردیابی وضعیت پرینت  

## فرمت‌های پشتیبانی شده

- **عکس**: JPG, JPEG, PNG
- **اسناد**: PDF, DOC, DOCX
- **متن**: TXT

## الزامات

- Android 7.0 (API 24) یا بالاتر
- کابل USB OTG
- پرینتر سازگار با USB

## نصب و اجرا

### با Android Studio

1. پروژه را در Android Studio باز کنید
2. Gradle synchronize کنید
3. پروژه را build و run کنید

### بدون Android Studio

```bash
./gradlew build
./gradlew installDebug
```

## استفاده

1. اپلیکیشن را باز کنید
2. دکمه "انتخاب فایل" را بزنید
3. فایل مورد نظر را انتخاب کنید
4. پرینتر را از طریق USB OTG متصل کنید
5. دکمه "پرینت کن" را بزنید

## معمارری

- **MainActivity.kt**: صفحه اصلی و کنترل رابط کاربری
- **PrinterService.kt**: سرویس USB و منطق پرینت
- **UsbPrinterReceiver.kt**: گیرنده‌ی رویدادهای USB

## مجوزهای مورد نیاز

- `READ_EXTERNAL_STORAGE`: خواندن فایل‌های دستگاه
- `WRITE_EXTERNAL_STORAGE`: نوشتن فایل‌های দাِستگاه  
- `USB_PERMISSION`: دسترسی به دستگاه‌های USB
